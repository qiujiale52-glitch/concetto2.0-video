# Concetto 2.0 Video

这是 Concetto 2.0 宣发视频项目的轻量化版本管理仓库。

仓库主要保存：

- 视频制作脚本与版本迭代脚本；
- 内容结构、素材取用、正片生成说明等文档；
- 少量关键前置视觉素材，例如开头、亮点、更新内容、九大环节总览图；
- Git/LFS 配置。

仓库不直接保存：

- 原始完整素材包；
- 剪映/宣发工程的大型资源目录；
- 音乐库、字体库等授权素材库；
- 渲染缓存、超分中间文件、逐帧缓存；
- 正片成片 mp4。

这些大文件建议继续保存在本地项目目录、网盘或 GitHub Release 中，避免污染 Git 历史。
