# Concetto 2.0 发布感样片 v1

时长：约 52 秒｜画幅：16:9｜定位：低清方向样片

## 这版的创作判断

原版有足够的功能与产品画面，但信息以清单式连续铺开，重点平均、节奏平，难以形成“2.0 登场”的记忆点。v1 不扩写原片没有证明的功能，只重组现有可信画面；用“全链路连接”作为统一叙事线。

## 结构

| 时间 | 段落 | 屏幕文案 | 原版画面 |
| --- | --- | --- | --- |
| 0–7s | 登场 | `CONCETTO 2.0` / `让设计的全链路，重新连接` | 0–8s |
| 7–15s | 场地与策略 | `CONNECT` / `场地、数据与方案，在同一处汇聚` | 22–30s |
| 15–23s | 空间与模型 | `MODEL` / `从空间构思，到可编辑的三维表达` | 34–42s |
| 23–32s | 图像表达 | `CREATE` / `让灵感与表达，保持同一种语言` | 60–69s |
| 32–41s | 方案深化 | `DESIGN` / `支持 DWG 协同，继续推进设计` | 90–100s |
| 41–49s | 全链路汇流 | `ORCHESTRATE` / `从推演、表达，到落地` | 110–119s |
| 49–52s | 收束 | `CONCETTO 2.0` / `重塑设计的完整旅程` | 0–3s（复现） |

## 视觉与声音

- 深墨紫舞台底 + 产品画面前置，避免原版浅紫满屏造成的“PPT感”。
- 画面以轻微推近、短交叉溶解与节拍切换组织；不堆叠故障、粒子等泛科技效果。
- 使用现有授权音乐库中的科技氛围曲作临时配乐，仅用于方向验证。
